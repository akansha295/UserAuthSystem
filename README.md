# 🔐 UserAuthSystem

User Authentication System using Servlets, JDBC, MySQL and Apache Tomcat.

---

## 📌 Project Description
A simple Java web application that allows users to register and log in.  
It stores user details in a MySQL database and validates credentials during login using JDBC and Servlets.

---

## ✨ Features
- User Registration (username, password, email)
- User Login with database validation
- MySQL database connectivity using JDBC
- Secure queries using PreparedStatement
- Deployed and tested on Apache Tomcat 9
- Clean separation of frontend and backend

---

## 🛠 Technologies Used
- Java (Core + Servlets)
- JDBC
- MySQL
- Apache Tomcat 9
- HTML

---

## ⚙️ Application Flow
1. User registers through register form  
2. Data is stored in MySQL database  
3. User logs in through login form  
4. Credentials are validated from database  
5. Success or error message is displayed
